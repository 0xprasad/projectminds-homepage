## 🎯 About The Project

ProjectMinds is a modern, responsive website designed to help final-year engineering students find and purchase ready-to-submit academic projects. Built with React and Vite, it features a clean yellow and black theme inspired by Spendesk's design philosophy.

### Why ProjectMinds?

- ✅ **Complete Package** - Code, Report, PPT all included
- ⚙️ **Customizable Projects** - Tailored for your syllabus and department
- ⏱️ **Fast Delivery** - Ready within 3-5 days
- 💬 **Support Till Viva** - Guidance till your presentation day

## ✨ Features

- 🎨 **Modern UI/UX** - Clean, minimal design with smooth animations
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile
- 🚀 **Fast Performance** - Built with Vite for lightning-fast load times
- 🎯 **Easy Navigation** - Fixed navigation bar with smooth scrolling
- 🏫 **College Showcase** - Animated scrolling banner of trusted colleges
- 📂 **Project Categories** - Filter projects by domain (CSE, AI/ML, IoT, etc.)
- 💬 **WhatsApp Integration** - Direct contact via WhatsApp for inquiries
- 🎭 **Smooth Animations** - Intersection Observer for scroll animations
- 📊 **Projects Page** - Dedicated page with filtering functionality
- 🌐 **SEO Optimized** - Proper meta tags and semantic HTML

## 🛠️ Tech Stack

**Frontend:**
- [React](https://reactjs.org/) - JavaScript library for building user interfaces
- [Vite](https://vitejs.dev/) - Next generation frontend tooling
- [React Router](https://reactrouter.com/) - Declarative routing for React

**Styling:**
- CSS Modules - Scoped and modular CSS
- Custom CSS Variables - For consistent theming

**Animations:**
- [React Intersection Observer](https://github.com/thebuilder/react-intersection-observer) - Scroll animations
- CSS Animations - Custom keyframe animations

**Tools:**
- Git - Version control
- npm - Package manager
